Project Management Site:CollabCore

A platform to manage projects in a team

I developed a full-fledged project management site using React and Firebase services for authentication, database, storage, and hosting. This platform allows users to sign up, log in, and view the online status of other users. Once logged in, users can create new projects by providing details such as project name, due date, and category. They can also assign tasks to team members.

Responsive Design: CollabCore boasts a fully responsive design, ensuring an optimal viewing experience on various devices, including mobile phones and tablets. This responsive design enhances user engagement and accessibility.

Development: Each user has a personalized profile with a thumbnail image and name displayed in the users section. Users can leave comments on various projects, fostering collaboration and communication. The dashboard includes a top navbar that enables users to filter projects by category or view only those assigned to them, enhancing project organization and visibility.
The user interface is designed for ease of use, featuring an intuitive layout and responsive design. Additionally, the platform includes a feature where only the creator of a specific project can mark it as completed or delete it, ensuring project data integrity.

Deploymant Process: The project utilizes Firebase services for authentication, database, storage, and hosting. The deployment process is streamlined, leveraging Firebase's capabilities for efficient hosting and scaling. I established robust Firestore database rules to control access and ensure data security. Define rules that restrict unauthorized access and maintain data integrity within the project management environment.

Key Features:

  • User authentication and authorization
  • Project creation with details (name, due date, category)
  • Task assignment to team members
  • User profiles with thumbnails
  • Commenting on projects
  • Top navbar for filtering projects
  • Owner-exclusive completion and deletion of projects
